# 🏦 Bank Loan Risk Analysis & Approval Prediction

## 📌 Project Overview
This project analyzes bank loan applicant data to identify key factors influencing loan approval decisions and builds a machine learning model to predict loan status.

## 🚀 Objectives
- Perform Exploratory Data Analysis (EDA)
- Identify high-risk customer segments
- Analyze impact of credit history, income, and property area
- Build a Logistic Regression model for loan approval prediction

---

## 📊 Key Insights

### 🔹 Credit History is the Strongest Factor
- Approval rate with good credit history: ~79%
- Approval rate with poor credit history: ~8%

### 🔹 Property Area Impact
- Semiurban areas show highest approval rates
- Rural areas show lowest approval rates

### 🔹 Income Alone is Not a Strong Predictor
Income groups had similar approval rates unless combined with credit history.

---

## 🤖 Machine Learning Model

- Model Used: Logistic Regression
- Accuracy Achieved: ~79%
- Strong performance in predicting approved loans
